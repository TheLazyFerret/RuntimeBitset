# RuntimeBitset
## About

library whose purpose is to be a substitute for std::bitset, but whose assignment is done at runtime.

## Installation

No dependecy needed

## Usage

```cpp
#include "RuntimeBitset/RuntimeBitset.hpp"
```

Uses same (or very similar) interface to std::bitset
